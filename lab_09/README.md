# Lab 09

## On UNIX systems

Compile with

```sh
$ make
```

Run with

```sh
$ ./lab_09
```

Preferable to compile and run under UNIX systems

## On other OS

To compile under other OS do:

```sh
$ ... lab_09.c error.c func.c ft_atoi.c ft_strcpy.c
```

Also include header file:

```sh
$ ... -I header.h
```
